# Collection of Experiments about PLN

## Overview

This folder contains a number of experiments about PLN, mostly to gain
understanding about it.  It contains a few chaining experiments but
only in the context of PLN, for more indepth chaining experiments see
the [chaining repository](https://github.com/trueagi-io/chaining).

## Description

It contains the following subfolders:

- [early-pln](early-pln) early on experiments that are probably no
  longer relevant but still kept around just in case.
- [truth-value-formula](truth-value-formula) contains experiments on
  truth value formula calculations.
