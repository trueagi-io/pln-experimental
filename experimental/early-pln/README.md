# Early PLN Chaining Experiments in MeTTa

## Overview

This code was written early on while still in process of learning
MeTTa.  It is not expected to be useful anymore but kept because some
pieces of code here and there may still be reused.

## Description

It contains the following subfolders:

- [synthesis](synthesis): program synthesis experiments (backward chaining).
- [common](common): some reusable type and function definitions.
- [pln](pln): collection of PLN port attempts.
